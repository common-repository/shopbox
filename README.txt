== Description ==

Shopbox and Woocommerce works together to always keep you webshop and physical shop working together.

So you can rest assure that you don't sell what you don't have and can easy manages both you webshop and store,
See feature list here:

Product sync. between both webshop and POS, to ensure easy and smoothly product and shop manage

Inventory sync, so you never sell something not in stock.

Requires

"Requires at least" is the lowest version that the plugin will work on

"Tested up to" is the highest version that you've successfully used to test the plugin. Note that it might work on
higher versions... this is just the highest one you've verified.

Stable tag should indicate the Subversion "tag" of the latest stable version, or "trunk," if you use /trunk/ for
stable.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/shopbox` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin
1. Create an api token from your shopbox account
2- go to shopbox plugin settings
3- add api token
4- choose branch, cash register to connect to.
